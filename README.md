# 🛒 Shopping Mall Management System (Python)

## 📌 Description

This is a command-line based shopping mall system built using Python.
It allows users to select a category, choose products, and generates a bill with automatic pricing.

---

## 🚀 Features

* Category selection (Men, Women, Children)
* Product selection based on category
* Size selection for clothing items
* Multiple payment options (Card, UPI, Cash)
* Automatic price calculation using predefined prices
* Bill generation

---

## 🛠️ Technologies Used

* Python

  * Conditional statements (`if-elif`)
  * Dictionaries
  * User input handling

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Run the Python file:

```bash
python shopping.py
```
